◆ recupererCollectNAS()

recupererCollectNAS ( $ftp_server,
$ftp_user,
$ftp_pass,
$URI_VIDEOS_A_ANALYSER,
$COLLECT_NAS,
$URI_NAS_RACINE )

Fonction qui récupère l'ensemble des métadonnées techniques des vidéos d'un NAS (collectPAD ou collectARCH) On télécharge les vidéos dans un $URI_VIDEOS_A_ANALYSER si celles-ci ne sont pas présentes dans la BD.

  • On remplit CollectNAS pour chaque vidéo
  • On vide le répertoire local $URI_VIDEOS_A_ANALYSER
    Paramètres
    ftp_server- Le nom du serveur ftp auquel on veut accéder
    ftp_user- Nom de l'utilisateur qui se connecte sur le serveur ftp
    ftp_pass- Mot de passe de l'utilisateur se connectant sur le serveur ftp
    URI_VIDEOS_A_ANALYSER- - URI de la vidéo qui doit être analysée
    COLLECT_NAS- Toutes les vidéos collectées sur les NAS
    URI_NAS_RACINE- URI de la vidéo sur le NAS racine

remplirCOLLECT_STOCK_LOCAL ( & $COLLECT_PAD,

$COLLECT_STOCK_LOCAL )

Fonction qui remplit $COLLECT_STOCK_LOCAL avec les metadonnées de chaque vidéo présentes dans $COLLECT_PAD ET $COLLECT_ARCH.

  • Vide les vidéos de $COLLECT_PAD et $COLLECT_ARCH qui sont ajoutées dans $COLLECT_STOCK_LOCAL (passage les collections par référence) Traite les vidéos isolées

    COLLECT_STOCK_LOCAL- Tableau qui sera rempli de données à l'issue de la fonction
    Renvoie
    COLLECT_STOCK_LOCAL - Tableau contenant toutes les vidéos qui doivent être stockées sur le serveur local
